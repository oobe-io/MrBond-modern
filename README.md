# MrBond-modern

Mr.Bond（九州工業大学製ボンドグラフ・シミュレータ）を、現代的なクロスプラットフォーム技術で再実装するプロジェクト。

## 背景

- 原作: Mr.Bond ver 1.5.2（九州工業大学）、32bit Windows 専用ネイティブアプリ
- 原作者が既に逝去されており、保守困難になっている
- 田中教授（九州工業大学）からの依頼で、現代環境でも動作する後継版を開発

## 目標

- BGEファイル（原作のモデル形式）との**互換読み込み**
- 原作と**数値的に一致する**シミュレーション結果
- Webブラウザで動作（OS非依存・インストール不要）
- ソルバ・パーサをテスト可能な純粋関数として構築

## ステータス

**解析フェーズ完了、実装フェーズ初期（ソルバ移植中）**

## 参照実装

オリジナルソースコードは Google Drive 上に保管（非公開）。
`Runge.f`（Fortran 90版）を参照実装として忠実移植を行う。C版 `Runge.c` には制約ソルバ（SOLV）内に齟齬があるため採用しない。

## アルゴリズム

ソルバは **Runge-Kutta-Gill 4次**（係数 `CS1 = 1/√2`, `CS2 = √2` を使用する丸め誤差軽減型）。詳細仕様は別途ドキュメント化。

## 開発

```bash
npm install
npm run typecheck
npm test
```
